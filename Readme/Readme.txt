・作品の名前：生徒情報開示システム
・作品の説明
家庭教師として働く先生へ向けたWebアプリをローカル環境で作成しました。
初訪問時に先生が使用するということを想定し、入力した生徒のIDとパスワードを
DB上のデータと照合し、IDとパスワードに紐づいた生徒の情報を開示できる仕組みです。

・作成した動機
学生時代、家庭教師のチューター(正規の先生が決まるまでの中継ぎ、訪問営業)をしていました。
チューターは固定の生徒ではなく、通常1回切りの指導を行うので、事前に生徒情報を確認する必要があります。
社内では指導生徒の情報をチューターに対して、エクセルデータを送り続けている社員がいました。
その社員は職場でPCに張り付いて送り続ける仕事の負担が大きく、生徒情報のアップデートまで手を回せていませんでした。
そこで、負担を軽減し労力を分散させ、生徒情報のアップデートに手を回せるようにする、ということを想定し
このウェブアプリを作成しました。

・開発環境
Eclipse IDE (4.22.0)
Server：Apache Tomcat/9.0.56
pgAdmin4 (PostgreSQLベースのデータベース)

・動作させるための準備
1. Studentという名前のDBを登録、テーブルを登録するために、ReadmeフォルダのSQL文に従ってクエリを実行する。
2. プロジェクトとDBを紐づける。webapp内のlibフォルダにpostgresql.jarを入れ、ビルドする。
3. java/service/StudentServiceを開き、15~18行を自身の環境に合わせて変更する。

・操作方法
1. webappフォルダ内にあるlogin.htmlをサーバーで実行する。
2. ログイン画面が表示されるので、
　 ID ：00001 、00002、00003 の3つのIDのうち1つを入力。
3. PASSWORDはadminを入力
4. ログインボタンを押す
5. result.jspへ遷移するので、確認する。
